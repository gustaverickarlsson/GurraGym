# GurraGym

GurraGym is currently a production vanilla PWA (`index.html` + `app.js`) with an in-progress modernization foundation in `src/`.

## Legacy app
- Entry: `index.html`
- Logic: `app.js`
- Styling: `style.css`

## Modernization foundation
- Domain modules: `src/domain`
- React + TypeScript app shell: `src/main.tsx`, `src/App.tsx`, `modern.html`
- Unit tests: `tests/unit`

## Commands
- `npm install`
- `npm run lint`
- `npm test`
- `npm run dev`
